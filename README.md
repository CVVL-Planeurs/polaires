# polaires

## Présentation

Ce projet fournit des outils pour le calcul et la visualisation des polaires de planeurs. Il permet à l'utilisateur de sélectionner un modèle de planeur, d'ajuster différents paramètres de vol et environnementaux, et de visualiser à la fois les résultats numériques et les courbes polaires interactives. Ce logiciel s'adresse principalement aux pilotes de planeur, instructeurs et passionnés d’aviation désireux d’analyser la performance de leur appareil.

## Fonctionnalités

- **Base de données de planeurs** : Liste complète de modèles de planeurs avec leurs données de polaire (taux de chute, vitesses, envergures, et autres paramètres).
- **Ajustement des paramètres** : L'utilisateur peut saisir ou modifier les paramètres qui influencent la courbe polaire, tels que la vitesse, le ballast, le vent, etc.
- **Calculs des polaires** : Calcul des principaux indicateurs de performance, comme le taux de chute minimal, la vitesse de meilleur plané, etc. pour le planeur sélectionné et les paramètres donnés.
- **Graphiques interactifs** : Visualisation des courbes polaires (taux de chute en fonction de la vitesse) et mise en avant des points importants, comme la finesse maximale.
- **Tableaux de résultats** : Présentation claire des résultats calculés.
- **Interface utilisateur** : Développée avec React et TypeScript, pour une expérience web moderne et réactive.

## Utilisation

1. **Sélectionner un planeur** : Choisissez un modèle dans la base de données via l’interface.
2. **Définir les paramètres** : Modifiez les paramètres de vol pertinents.
3. **Afficher les résultats** : Consultez les indicateurs calculés et les courbes polaires interactives.
4. **Analyser les données** : Utilisez les résultats pour la préparation du vol, l’instruction ou la comparaison de performances.

## Technologies

- **TypeScript** et **React** pour la logique et l’interface.
- **MUI (Material UI)** pour les composants d’interface et les graphiques.
- **Vite** pour le développement et la compilation.
- **CSS/HTML** pour la mise en forme et la structure statique.

## Avertissement

Les courbes polaires sont des modélisations mathématiques basées sur les données disponibles ; les valeurs extrêmes sont des approximations. Toujours vérifier avec la documentation officielle de votre appareil et considérer cet outil comme une aide complémentaire.
